INTRODUCTION:
-----------------------------------------------------------------
This is the top-level git repository for the DES LSS group for code development. 

DISTRIBUTION:
-----------------------------------------------------------------
To checkout this module, you need one of two things:
1) be kerberized at fermilab
2) have your public ssh key added to the database

For (1), you can follow the directions here:
https://cdcvs.fnal.gov/redmine/projects/desecs/wiki/Getting_Started#Fermilab-Kerberos-Account
For (2), for now you will need to contact Nacho at nsevilla@gmail.com

Once you have access, you can check out the full repository
with the command:
> git clone ssh://p-des-lss@cdcvs.fnal.gov/cvs/projects/des-lss-papers

Or, you can check out only a submodule (for example, the Y1 BAO sample paper)
with the command:
> git clone ssh://p-des-lss@cdcvs.fnal.gov/cvs/projects/des-lss-papers/y1baosample_paper

CREATION:
-----------------------------------------------------------------
If you would like to have a new repository created, please contact
the project managers:
https://cdcvs.fnal.gov/redmine/projects/des-lss

Redmine supports three version control cvs, svn, and git. More
details on repository creation can be found here:
https://cdcvs.fnal.gov/redmine/projects/fermi-redmine/wiki/CreatingRepositories

Documentation on version control tools can be found on Google:
cvs - http://www.nongnu.org/cvs/
svn - http://svnbook.red-bean.com/
git - http://git-scm.com/
